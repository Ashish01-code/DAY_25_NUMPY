### 2. NumPy Array Operations
Performed element-wise operations using NumPy ufuncs:
- Addition (`np.add`)
- Subtraction (`np.subtract`)
- Multiplication (`np.multiply`)
- Power operations (`np.power`)
- Remainder (`np.remainder`)
- Quotient and remainder (`np.divmod`)
- Absolute values (`np.absolute`)

These operations demonstrate **vectorized computation**, which replaces traditional loops in ML pipelines.
